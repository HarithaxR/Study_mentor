# 📚 AI Study Mentor

AI Study Mentor is a web-based learning assistant designed to help students understand concepts more effectively through explanations, summaries, key points, and question-based learning.
This project focuses not only on functionality but also on demonstrating how modern AI tools like ChatGPT and Claude can assist in building real-world applications.


## Features

*  Concept explanation (simple & clear)
*  Summary generation
*  Key point extraction
*  Question generation


##  Tech Stack

* React.js
* HTML, CSS, JavaScript
* Node.js
* OpenAI (ChatGPT)
* Claude AI (Anthropic)


##  How AI Tools Were Used

### 🔹 ChatGPT

* Helped in generating:
  * UI design ideas
  * Code structure (React components, styling)

---

### 🔹 Claude AI

* Used for:
  * Improving user interface design
  * Creating clean and readable code
  * Improving content clarity
  * Helping with logical flow of features
